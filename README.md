## 🧭 Sobre mim

Venho de um caminho não linear.

Antes da tecnologia, vieram outras vivências: trabalho manual, roça, construção, música, rádio e comunidade.  
Foi nesse percurso que encontrei a tecnologia — não como ponto de partida, mas como ferramenta de transformação.

Hoje atuo como desenvolvedor Full Stack, criando e mantendo sistemas, APIs e produtos digitais — sempre com foco em resolver problemas reais.

Além disso, construo projetos próprios e compartilho o processo, não só o resultado.

> Quem vê o código rodando não vê o caminho até ele.
